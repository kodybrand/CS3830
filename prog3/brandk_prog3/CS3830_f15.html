Course Description:
An introduction to data communications and computer networks. Study of the basic principles, with a focus on the layers, architectures, protocols, and security used in the Internet. Some programming projects involving sockets.

Course Outcomes:

  • Describe common network application protocols including email, telnet, ftp, and http.
  • Explain the layered structure of TCP/IP protocol.
  • Describe the purpose of each layer of TCP/IP network.
  • Understand the various protocols inside TCP/IP protocol suite.
  • Understand how Internet routing works.
  • Understand basic techniques used for secure communications.
  • Write programs using sockets to communicate.
